# C++ `sort` 函数超详细介绍（编程必用、本题核心）
`sort` 是 C++ **标准库最常用的排序函数**，效率高、写法简单、稳定可靠，本题**必须用它**才能拿满分。

---

## 一、基本信息
1. **头文件**
```cpp
#include <algorithm>
```
2. **功能**
对一段数据**默认升序排序**（从小到大）
3. **底层原理**
快速排序优化版（introsort），时间复杂度 **O(n log n)**
4. **效率极高**
n=1e5 轻松秒过，比你写的插入排序快 1000 倍以上

---

## 二、最常用写法（本题用这个）
```cpp
sort(起始地址, 结束地址);
```

### 例子 1：vector 排序
```cpp
vector<int> arr = {3,1,2};
sort(arr.begin(), arr.end());
// 排序后：1 2 3
```

### 例子 2：普通数组排序
```cpp
int a[] = {5,3,8};
sort(a, a+3);
// 排序后：3 5 8
```

---

## 三、降序排序（从大到小）
```cpp
sort(arr.begin(), arr.end(), greater<int>());
```

---

## 四、`sort` 的范围规则（重要！）
**左闭右开**
```cpp
sort(begin, end);
```
- 包含 `begin`
- **不包含** `end`

和 `accumulate`、`lower_bound` 规则完全一样。

---

## 五、为什么本题必须用 sort？
你原来用的：
```cpp
lower_bound + insert
```
是**插入排序**，时间复杂度 O(n²)
n=1e5 时会直接**超时爆掉**。

`sort` 是 O(n log n)，**大数据必用**。

---

## 六、最简单记忆口诀
```
sort(头, 尾) → 从小到大排
sort(头, 尾, greater<int>()) → 从大到小排
```

---

## 七、本题中 sort 的作用
1. 把活跃度**从小到大排序**
2. 前一半 → 内向（低活跃度）
3. 后一半 → 外向（高活跃度）
4. 差值最大（贪心算法）

---

### 总结
- `sort` = C++ 万能排序
- 头文件 `<algorithm>`
- 写法：`sort(开始, 结束)`
- 默认**升序**
- 效率极高，本题必用
