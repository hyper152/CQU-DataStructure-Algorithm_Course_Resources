# 已知遍历序列求层序遍历

## 问题背景

给定一棵二叉树的一种或多种遍历序列，还原出树的层序遍历结果。

常见题型：

1. **完全二叉树 + 后序/前序/中序 → 层序**（如 PTA 7-7）
2. **一般二叉树 + 中序+前序 或 中序+后序 → 层序**

---

## 1. 完全二叉树（后序 / 前序 / 中序 → 层序）

### 原理

完全二叉树可以用 **1-indexed 数组**（堆式存储）唯一表示：

- 根节点下标为 `1`
- 节点 `i` 的左孩子为 `2*i`，右孩子为 `2*i+1`
- 数组下标顺序就是层序遍历顺序

对数组下标做对应顺序的遍历，同时按序从给定序列中取值填入数组即可还原。

以 **后序→层序** 为例：

```
输入后序: 91 71 2 34 10 15 55 18

数组下标（层序顺序）:
       1
     /   \
    2     3
   / \   / \
  4   5 6   7
 /
8

后序遍历下标顺序: 8→4→5→2→6→7→3→1
依次填入后序序列: 91 71 2 34 10 15 55 18

结果数组（层序）: [18, 34, 55, 71, 2, 10, 15, 91]
```

### 代码实现

三种遍历的差异仅在递归函数中 **赋值与递归调用的顺序**：

```python
def solve():
    n = int(input())
    seq = list(map(int, input().split()))  # 给定的遍历序列
    tree = [0] * (n + 1)
    idx = 0

    # 后序版: 左 → 右 → 根
    def post_fill(i: int):
        nonlocal idx
        if i > n:
            return
        post_fill(2 * i)
        post_fill(2 * i + 1)
        tree[i] = seq[idx]  # 最后赋值根
        idx += 1

    # 前序版: 根 → 左 → 右
    def pre_fill(i: int):
        nonlocal idx
        if i > n:
            return
        tree[i] = seq[idx]  # 先赋值根
        idx += 1
        pre_fill(2 * i)
        pre_fill(2 * i + 1)

    # 中序版: 左 → 根 → 右
    def in_fill(i: int):
        nonlocal idx
        if i > n:
            return
        in_fill(2 * i)
        tree[i] = seq[idx]  # 中间赋值根
        idx += 1
        in_fill(2 * i + 1)

    # 按需选择: post_fill(1) / pre_fill(1) / in_fill(1)
    post_fill(1)
    print(' '.join(map(str, tree[1:])))
```

## 2. 一般二叉树（中序+前序 / 中序+后序 → 层序）

### 原理

- **中序 + 前序**：前序第一个节点是根，在中序中找到根的位置划分左右子树
- **中序 + 后序**：后序最后一个节点是根，在中序中找到根的位置划分左右子树

建树后 BFS 输出层序。

### 代码实现

两种重建方式的差异仅在 **根节点的选取位置**：

```python
from collections import deque

class TreeNode:
    def __init__(self, val=0):
        self.val = val
        self.left = None
        self.right = None

# 前序 + 中序 重建
def build_pre_in(preorder, inorder):
    if not preorder or not inorder:
        return None
    root_val = preorder[0]                   # 前序第一个为根
    root = TreeNode(root_val)
    idx = inorder.index(root_val)            # 在中序中找到根的位置，左边为左子树，右边为右子树
    root.left = build_pre_in(preorder[1:1 + idx], inorder[:idx])
    root.right = build_pre_in(preorder[1 + idx:], inorder[idx + 1:])
    return root

# 中序 + 后序 重建
def build_in_post(inorder, postorder):
    if not inorder or not postorder:
        return None
    root_val = postorder[-1]                 # 后序最后一个为根
    root = TreeNode(root_val)
    idx = inorder.index(root_val)
    root.left = build_in_post(inorder[:idx], postorder[:idx])
    root.right = build_in_post(inorder[idx + 1:], postorder[idx:-1])
    return root

def level_order(root):
    if not root:
        return []
    res = []
    q = deque([root])
    while q:
        node = q.popleft()
        res.append(node.val)
        if node.left:
            q.append(node.left)
        if node.right:
            q.append(node.right)
    return res

def solve():
    n = int(input())
    seq1 = list(map(int, input().split()))  # 前序 或 中序
    seq2 = list(map(int, input().split()))  # 中序 或 后序

    # 按需选择:
    # root = build_pre_in(seq1, seq2)   # 输入为 前序+中序
    # root = build_in_post(seq1, seq2)  # 输入为 中序+后序
    root = build_pre_in(seq1, seq2)
    print(' '.join(map(str, level_order(root))))
```

## 总结

| 题型 | 核心思路 | 复杂度 |
|------|----------|--------|
| 完全二叉树 + 后序 → 层序 | 对下标后序遍历，按序填入 | O(N) |
| 完全二叉树 + 前序 → 层序 | 对下标前序遍历，按序填入 | O(N) |
| 完全二叉树 + 中序 → 层序 | 对下标中序遍历，按序填入 | O(N) |
| 中序 + 前序 → 层序 | 前序首元素为根，递归建树 + BFS | O(N²) / O(N)（哈希优化） |
| 中序 + 后序 → 层序 | 后序末元素为根，递归建树 + BFS | O(N²) / O(N)（哈希优化） |

> 注：一般二叉树的建树可以用哈希表预处理中序下标，将 `index()` 的 O(N) 降为 O(1)，整体降至 O(N)。
